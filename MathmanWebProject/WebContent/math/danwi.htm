단위 분수의 분해
날 언제까지 분해할 거죠? 있는 그대로 봐 줄 순 없나요?

    1. 정의

분자가 1, 분모가 2 이상인 분수를 단위분수라 하자.
그러면 하나의 단위분수는 n(2 이상) 개의 단위 분수의 합으로 나타낼 수 있다.
모든 단위분수에 대해 이를 두 개의 단위분수의 합으로 나타낼 수 있으면 되는데, 이 방법이 있기 때문이다.
여기서는 단위분수의 분해에 관한 재밌는 성질을 이야기 하고자 한다.

하나의 단위분수(1/a)를 두 개의 단위분수의 합(1/b+1/c)으로 나타낼 수 있을 때, 이에 대한 일반식을 다음과 같이 표현하자.

이 경우 1/a를 1/b와 1/c로 2단분해했다고 하자. 또한, 이 때, 이를 만족하는 해를 순서쌍 (a,b,c)으로 나타내자.
앞으로 위 조건을 만족하는 a,b,c에 대해 다루기로 한다.

    2. 단위분수의 2단분해

위와같은 단위분수의 2단분해에는 다음과 같은 재밌는 성질이 있다.

  • b의 범위는 항상 a보다는 크고 2a보다는 작다.
  • 모든 a에 대해 a와 서로소인 b를 잡으면 c는 항상 유일하다.
  • 또한 이 때, b=a+1, c=a(a+1)이다.
  • a,b,c가 모두 서로소인 해 (a,b,c)는 존재하지 않는다.
위 성질들은 조금만 생각하면 쉽게 증명할 수 있으리라 생각한다.

    3. 다단분해의 예

 2단분해의 해를 적당히 치환하면 다단분해의 해를 만들 수 있다.
그 몇 가지 예들을 적어보면 다음과 같다.

3단분해의 예

    1/2 = 1/3 + 1/7 + 1/42
    1/2 = 1/4 + 1/5 + 1/20
    1/2 = 1/4 + 1/6 + 1/12
    1/3 = 1/4 + 1/13 + 1/156
    1/3 = 1/5 + 1/12 + 1/20
    1/3 = 1/6 + 1/7 + 1/42
    1/4 = 1/6 + 1/20 + 1/30
    1/5 = 1/7 + 1/30 + 1/42
4단분해의 예
    1/2 = 1/4 + 1/7 + 1/12 + 1/42
    1/3 = 1/6 + 1/12 + 1/20 + 1/30
    1/4 = 1/7 + 1/20 + 1/30 + 1/42
    1/5 = 1/8 + 1/30 + 1/42 + 1/56
5단분해의 예
    1/2 = 1/5 + 1/7 + 1/12 + 1/20 + 1/42
    1/3 = 1/7 + 1/12 + 1/20 + 1/30 + 1/42
    1/4 = 1/8 + 1/20 + 1/30 + 1/42 + 1/56
    1/5 = 1/9 + 1/30 + 1/42 + 1/56 + 1/72
이와같이 얼마든지 만들 수 있다.
